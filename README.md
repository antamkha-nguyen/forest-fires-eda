# Forest Fires EDA (UCI ML Repository)

This project performs exploratory data analysis on the Forest Fires dataset from the UCI Machine Learning Repository using Python.

## Project Overview
- Dataset: UCI Forest Fires (Portugal)
- Records: 517 observations
- Focus: Weather conditions, seasonality, and burned area behavior

## What This Project Covers
- Reproducible data loading using the `ucimlrepo` package
- Data quality checks and summary statistics
- Distribution and relationship analysis of weather variables
- Seasonal analysis by day of the week and month of the year
- Exploration of the burned area variable using a log transformation
- Correlation analysis and interpretation

## Tools Used
- Python
- pandas, numpy
- matplotlib, seaborn
- Jupyter Notebook (VSCode)

## Notes
This project focuses on exploratory data analysis rather than predictive modeling.
